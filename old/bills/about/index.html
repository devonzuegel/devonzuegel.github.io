About

This Congressional Bill Outcome Predictor was originally a project for Stanford's CS 221 class entitled "Artificial Intelligence: Principles and Techniques" taught by Percy Lang. You can find the project code and pointers to the data on Github. Our group included Arushi Jain, Emma Marriott, and Devon Zuegel.

Feel free to contact us if you have any questions or comments about the project.
